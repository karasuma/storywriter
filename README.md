# Storywriter
いろんなののストーリーとか編集したりするやつ

## Commands
```
// Prepare
$> yarn install

// Execute
$> yarn electron:serve

// Build (windows)
$> build-action.bat

// Build (any)
$> yarn electron:build
$> cp default.ysd ./dist_electron/[any directories]/

// Test
$> yarn jest:unit
// not "t"est:unit
```
